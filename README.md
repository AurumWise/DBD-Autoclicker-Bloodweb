## ⚠️ ВАЖНО: ПРОГРАММА УПРАВЛЯЕТ МЫШЬЮ

DBD Автокликер Bloodweb может **перемещать курсор и выполнять клики** по результатам локального распознавания изображений.

Распознавание изображений не бывает безошибочным. Неверно настроенная сетка, изменение масштаба интерфейса, всплывающие подсказки, перекрытия окон, обновления игры или ошибки распознавания могут привести к тому, что программа определит объект неверно или попытается нажать не туда.

Используя программу, Вы подтверждаете, что:

- запускаете её добровольно и под личным контролем;
- понимаете, что программа может ошибаться;
- самостоятельно проверяете правильность сетки, шаблонов и очереди перед запуском;
- принимаете все риски, связанные с использованием программы;
- не предъявляете автору претензии за ошибки распознавания, неверные клики, потерю прогресса, игровые последствия, блокировки, ограничения, повреждение данных или иной прямой/косвенный ущерб.

Программа распространяется **«как есть»** (`AS IS`), без гарантий бесперебойной, безопасной, точной или подходящей для конкретной цели работы.

## Что делает программа

DBD Автокликер Bloodweb работает только через обычный пользовательский интерфейс:

- делает локальные скриншоты экрана;
- использует локальное компьютерное зрение для поиска изображений в ROI узлов Bloodweb;
- показывает сетку, которую пользователь вручную совмещает с Bloodweb;
- использует приоритетные предметы и выбранный шаблон;
- строит очередь будущих кликов;
- умеет запускать **тест** без кликов;
- после явного нажатия `Start` временно двигает мышь и выполняет клики.

Проект **не читает память игры**, не внедряет DLL, не изменяет игровые файлы, не использует игровые API и не анализирует сетевой трафик игры.

## Как это работает

1. Откройте Bloodweb в игре.
2. Запустите DBD Автокликер Bloodweb.
3. Настройте центр Bloodweb — это защитный визуальный якорь.
4. Покажите и совместите сетку с узлами Bloodweb.
5. Выберите шаблон и приоритетные предметы.
6. Нажмите `Тест`, чтобы увидеть предполагаемую очередь без кликов.
7. Проверьте очередь.
8. Нажмите `Start` только когда всё выглядит правильно.

Программа сначала ищет предметы из списка **«Приоритет»**, затем предметы выбранного шаблона.

## Защитные механизмы

### Центр Bloodweb

Перед автокликом и во время работы программа проверяет эталонный центр Bloodweb.

Индикатор центра:

- 🟢 **Центр распознан** — защита подтверждена, клики могут быть разрешены.
- 🟡 **Перепроверка центра** — новые клики временно заблокированы.
- 🔴 **Центр не распознан** — автоклик заблокирован.
- ⚪ **Центр не настроен** — `Start` недоступен.

Логика безопасности:

- после запуска нужно получить **две успешные проверки центра подряд**;
- если центр временно не найден, новые клики сразу блокируются;
- затем выполняются две быстрые повторные проверки;
- если центр не подтверждён, автокликер останавливается;
- после полноценной остановки программа **не продолжает работу сама** — для нового запуска требуется вручную нажать `Start`.

> **Нет подтверждённого центра Bloodweb — нет автокликов.**

### Парковка курсора

Стандартный цикл выглядит так:

```text
парковка курсора
→ переход к цели
→ задержка перед кликом
→ клик
→ немедленный возврат на парковку
→ задержка после клика уже вне предмета
```

Это уменьшает риск появления hover-подсказок и перекрытия областей, используемых для распознавания.

### Аварийная остановка

Автокликер должен быть остановлен немедленно при любом из условий:

- нажата кнопка **`Stop · F8`**;
- нажата глобальная горячая клавиша **`F8`**;
- пользователь вручную сдвинул мышь;
- центр Bloodweb перестал подтверждаться;
- произошла ошибка скриншота, распознавания или внутренней логики.

После остановки очередь отменяется, новые клики запрещаются, а причина показывается в интерфейсе и логе.

## Если что-то пошло не так

### Центр не распознан

1. Не нажимайте `Start`.
2. Уберите всплывающие подсказки и посторонние окна.
3. Убедитесь, что открыт именно Bloodweb.
4. Проверьте масштаб интерфейса игры и положение окна.
5. При необходимости перезапишите эталон центра через кнопку `Центр`.

### Сетка не совпадает с узлами

1. Нажмите `Паутина`.
2. Совместите и масштабируйте сетку вручную.
3. Снова сделайте `Тест`.
4. Не запускайте `Start`, пока очередь не выглядит корректной.

### Программа остановилась

Это нормальное защитное поведение, если:

- Вы сдвинули мышь;
- нажали `F8`;
- центр временно пропал;
- произошла ошибка распознавания.

Проверьте статус, исправьте причину и только потом запускайте новый цикл вручную.
